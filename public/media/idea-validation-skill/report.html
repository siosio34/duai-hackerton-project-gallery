    1왜 지금 이 아이디어인가

    2024년 한국인 해외 출국자는 약 2,868만 명으로 코로나 이전 수준을 완전히 회복했고, 20·30대(연 986만 명 출국)는 해외여행 선호도가 가장 높은 세그먼트입니다. 친구·연인 동행 비율이 약 33%, 그룹 조율 채널은 사실상 카카오톡(국내 점유율 ~97%) 단톡방입니다. "단톡에 링크는 쌓이는데 정작 여행 가면 아무도 못 찾는다"는 페인은 실재합니다. LLM 기반 링크·장소 추출 기술도 성숙 단계입니다.

    그러나 타이밍이 좋다는 것과 사업이 된다는 것은 다릅니다. 아래 네 가지가 발목을 잡습니다.

    2시장 평가 — 원본 55 / 피벗 63 (100점)

    두 번의 패스로 평가했습니다. 1차(요약 패스)는 원본을 보수적으로 38점, 2차 심층 패스는 원본 55 / 피벗 63점으로 재채점했습니다(문제 증거를 더 높게 인정). 아래는 심층 패스 기준이며, 결론(피벗 권고)은 양쪽 동일합니다.

    2-4. 화이트스페이스 · 방어가능성

    정확한 갭: ①어떤 출처든 링크 무마찰 수집 + ②실시간 그룹 협업 보드 + ③카카오맵 시각화 + ④카카오링크 네이티브 공유를, 20-30대 친구 해외여행에 최적화해 하나로 묶은 한국어 제품이 없음. "인스타에서 식당, 네이버블로그에서 숙소, Klook에서 액티비티 찾은 걸 친구 3명과 모아 함께 일정 만든다"는 행위가 한국에서 미충족.

    방어가능성: 중간(조건부). ① 트리플이 1-2스프린트면 URL임포트로 갭을 닫을 수 있음 ② 진짜 해자 후보는 카카오링크 바이럴 루프(아티팩트=초대장, Notion·Figma식 네트워크 효과) ③ 제휴 수수료는 OTA가 정해 단가 우위 없음. 윈도우 12-18개월 — 트리플이 URL임포트를 붙이기 전에 바이럴 루프를 완성하느냐가 관건.

    2-5. 지불의사 · 타이밍

    WTP: 한국 사용자는 플래닝 도구에 구독료를 내지 않음(국내 플래닝 앱 전부 무료, Wanderlog/TripIt Pro도 매출 비주력). 유효 모델은 OTA 제휴 수수료 — 마이리얼트립(투어 20%)·글로벌 OTA(숙박·항공 3-5%)로 검증된 원리. 단, TripLink 트래픽 품질·전환율은 미검증. 친구그룹은 예산 분산형이라 그룹 누구나 예약 클릭 가능 → 여행당 복수 제휴 전환 기회.

    타이밍 테일윈드: 출국 완전 회복(2024년 2,872만, 2019년의 99.4%) 후 확장기 · 20-30대 해외·친구동행 선호 최고 · 링크 파편화 가속(네이버·유튜브·인스타·Klook 동시 탐색) · AI가 일정생성을 무료 상품화 → TripLink는 AI가 못 대체하는 소셜·협업 레이어로 경쟁(올바른 선택) · 카카오링크 API 무료·성숙. 단 국내 여행앱 신규 설치는 2026 Q1 -23%(설치 시장 위축).

    2-7. 가장 중요한 미검증 증거

    "그룹 활성화율" — 실제 친구그룹(3-4인) 5-10팀 테스트에서, 생성된 트립보드 중 72시간 내 동행 2명 이상이 합류해 장소를 추가하는 비율. 이 한 지표가 바이럴 루프 작동/붕괴를 결정. 30% 미만이면 카카오링크 바이럴이 CAC를 못 이겨 유료획득 필요 → ARPU(연 ₩2-8만)가 한국 UA CPI(₩3천-1.5만)를 감당 못함. 50% 초과면 시장 신뢰도 MEDIUM→HIGH 승급.

    평결 reasoning: 페인 검증됨 + 합법적 아키텍처(붙여넣기) 존재 + 한국 네이티브 차별점 가용 + 아직 실험 미실시 → KILL은 시기상조. 단, ① 프로토타입 완주율<20% 또는 ② 공유 재진입 K<0.1 또는 ③ 네이버 og:메타 추출이 API 계약 없이 DB권 침해 판정 시 즉시 KILL로 전환.

    카운슬 판단은 증거 사다리 5단계(합성 판단). 법적 서술은 제품 설계 목적의 이슈 스팟팅이며 자문이 아님 — 베타 전 한국 변호사 검토 필요.

    6권고 — 피벗과 MVP

    핵심 피벗: 채팅 업로드 제거 → URL 붙여넣기 협업 보드

    사용자가 여행 링크를 직접 붙여넣으면 og:메타데이터+좌표를 추출해 지도 핀으로 보여주고, 보드를 카카오링크로 공유. 이 한 번의 설계 변경으로 PIPA·크롤링·신뢰 리스크가 동시에 크게 완화됩니다.

    • 지도: 구글맵 대신 카카오맵/네이버 지도 API(국내 장소 커버리지·ToS 우위)
    • 수익: 광고가 아닌 OTA 제휴 커미션 중심으로 재설계
    • 성장: 공유 보드 비주얼을 먼저 설계(카카오링크·인스타 스토리 1탭 공유 = 유일한 CAC 레버)
    • 2차 옵션: 카카오톡 채널 챗봇(링크 전달) / OTA B2B2C 화이트라벨

    MVP 범위 — 포함: URL 붙여넣기 → og:메타+좌표 → 카카오맵 핀 → 공유 보드. 제외: 채팅 .txt 업로드, 자동 동선 생성, 광고 배너.
