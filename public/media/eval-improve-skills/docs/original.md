# AI 자동매매 봇 기능 추가 기획서 (초안 / v0)

> 본 문서는 거래소에 "AI 자동매매 봇" 기능을 붙이기 위한 **대략적인 초안**이다.
> (의도적으로 거친 1차 버전 — 비교 데모의 baseline)

## 1. 배경
요즘 거래소 유저들이 자동매매를 많이 찾는다. 경쟁사도 비슷한 걸 내놨고 우리도 빨리 만들어야 한다.
AI가 알아서 사고팔아주면 유저가 좋아할 것 같다.

## 2. 목표
- 유저가 버튼 누르면 봇이 알아서 매매해준다.
- 수익이 나면 좋다.
- 신규 유저 유입과 거래량 증가.

## 3. 주요 기능
- 봇 켜기/끄기
- 전략 몇 개 제공 (예: 단타, 장기, DCA)
- AI가 시장 보고 알아서 판단
- 수익률 화면에 보여주기
- 알림

## 4. 사용자 흐름
1. 유저가 봇 메뉴 들어간다
2. 전략 고른다
3. 돈 넣는다
4. 봇이 돌아간다
5. 결과 본다

## 5. AI 모델
- 시세 데이터 넣고 살지 팔지 예측하는 모델 쓴다.
- 과거 데이터로 학습.
- 정확도 높이면 됨.

## 6. 일정
- 1차: 한두 달 안에 MVP
- 2차: 고도화

## 7. 기타
- 리스크는 유저 책임.
- 규제는 나중에 본다.
- 수수료로 돈 번다.
